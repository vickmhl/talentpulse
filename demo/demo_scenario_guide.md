# TalentPulse V7 演示说明

## 官方案例

建议用于：

1. 首页进入后的正式演示
2. 总览页 2-3 分钟讲述
3. 汇报报告页收口

推荐顺序：

1. 先看总览，给出公司现状、风险等级与当前优先动作
2. 再切部门现状，说明哪些团队强、哪些团队脆
3. 再点开三个核心问题，讲风险外推与建议动作
4. 补充组织运营页，用三年人数、离职、招聘与内部补位轨迹解释问题来源
5. 最后进入汇报报告，完成收口

## AI 清洗演示版

建议用于：

1. 上传演示
2. 数据健康页
3. 字段识别、自动归一与可信度说明

展示重点：

1. 系统自动识别常见字段含义
2. 系统自动统一部门、岗位、准备度等口径
3. 系统自动标记重复、异常、缺失与冲突
4. 在存在风险提示时，仍能说明哪些结论可以继续使用
