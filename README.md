# TalentPulse V8

TalentPulse 是一个面向 HR 学生与初级 HR 从业者的中文组织诊断作品。

它不是配置型后台，也不是普通分析看板，而是一套把员工数据转成可汇报组织判断、风险外推与建议动作的 PC 网页展示台。

核心主张：

`把员工数据转成可汇报的组织诊断。`

## V8 升级重点

V8 聚焦三件事：

1. 视觉与排版重构
2. 图表体系重构
3. Demo 数据模型重建

## 当前页面结构

- 首页
- 总览
- 组织运营
- 人才盘点
- 继任分析
- 数据健康
- 汇报报告

其中：

- 首页 / 总览 / 汇报报告：主讲页
- 组织运营 / 人才盘点 / 继任分析 / 数据健康：证据页

## 双演示样本

### 官方案例

- 用于首页、总览与汇报报告
- 数据口径统一，不故意注入脏值
- 附带 2023-01 至 2025-12 的组织、人力与招聘轨迹

### AI 清洗演示版

- 用于上传演示和数据健康页
- 保留字段别名、部门别名、岗位简称、缺失与异常
- 组织故事与官方案例一致

## 数据资产

当前 Demo 包含五层数据：

1. `employee_master`
2. `workforce_monthly`
3. `recruiting_reqs`
4. `exit_events`
5. `critical_roles`

## 三年演化主线

### 2023

- 扩张期
- headcount 增长快
- 招聘量高
- 研发和产品扩编
- 管理梯队开始跟不上

### 2024

- 冲刺期
- 销售业绩上来
- 关键人才流失率抬升
- 招聘成本上扬
- 客户成功与交付运营承压

### 2025

- 收敛期
- 总人数增速放缓
- 运营趋稳但停滞
- 支持职能与产品关键岗位暴露更明显
- 招聘周期回落，但内部供给仍不足
