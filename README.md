# TalentPulse V5

TalentPulse 是一个面向 HR 学生与初级 HR 从业者的中文组织诊断作品。

它不是配置型后台，也不是普通看板，而是一套把员工数据转成可汇报组织诊断的 PC 网页分析台。

核心主张：

`把员工数据转成可汇报的组织诊断。`

## 当前版本目标

V5 聚焦一件事：

让用户能按 `公司现状 -> 部门现状 -> 组织问题 -> 风险外推 -> 建议报告` 的顺序完成面试汇报。

## 六个主页面

- 首页
- 总览
- 人才盘点
- 继任分析
- 数据健康
- 汇报报告

其中：

- 首页 / 总览 / 汇报报告：主讲页
- 人才盘点 / 继任分析 / 数据健康：证据页

## 双演示样本

### 官方案例

- 用于首页、总览和汇报报告
- 数据口径统一，不故意注入脏值
- 组织故事稳定，可直接演示

### AI 清洗演示版

- 用于上传演示和数据健康页
- 保留字段别名、部门别名、岗位简称、缺失与异常
- 组织故事与官方案例一致

## 官方案例公司

公司名：`澄曜科技`

核心组织故事：

- 研发中心：高潜储备不低，但现在可接任管理梯队偏薄
- 销售增长：当前业绩强，但头部依赖重、保留风险高
- 交付运营：运行稳定，但成长动能不足、后备薄
- 产品与设计：关键岗位集中，单点依赖明显
- 客户成功：团队规模不小，但高潜识别不足
- 人力资源 / 财务法务 / IT与数据平台：人数少但继任暴露高

## 当前分析框架

当前引擎分两步完成诊断：

1. 识别结构信号
2. 组合成组织问题

结构信号包括：

- 高潜集中但现在可接任偏薄
- 高绩效集中且高流失风险高
- 关键岗位无后备聚集
- 覆盖存在但深度不足
- 单点依赖明显
- 稳定但低高潜、低 1-2 年可接任储备
- 支持职能低可见度风险

## V5 页面重点

### 首页

- 中文主张
- 两个入口：体验官方案例 / 上传员工数据
- 官方案例预览

### 总览

- 公司现状总判断
- 部门现状对比
- 三个核心问题
- 风险外推
- 建议动作
- 查看更多证据

### 人才盘点

- 结构判断
- 九宫格证据
- 部门现状摘要
- 伪强项
- 梯队薄弱点
- 人才样本抽屉

### 继任分析

- 岗位暴露结构
- 无后备 / 名义覆盖但弱 / 单点依赖
- 覆盖快照
- 支持职能风险
- 岗位与候选人样本抽屉

### 数据健康

- 可信度结论
- 自动识别了什么
- 自动修复了什么
- 需要谨慎的地方
- 影响范围
- 附录证据

### 汇报报告

- 公司现状
- 部门现状摘要
- 三个核心问题
- 根因解释
- 风险外推
- 优先建议

## 当前运行入口
