# TalentPulse V7

TalentPulse 是一个面向 HR 学生与初级 HR 从业者的中文组织诊断作品。

它不是配置型后台，也不是普通分析看板，而是一套把员工数据转成可汇报组织判断、风险外推与动作建议的 PC 网页展示台。

核心主张：

`把员工数据转成可汇报的组织诊断。`

## V7 升级重点

V7 聚焦两件事：

1. 高质感视觉与交互重构
2. 三年人力与招聘模拟数据重建

当前版本重点把产品从“静态分析 demo”升级成“高质感中文组织诊断展示台”。

## 当前页面结构

- 首页
- 总览
- 组织运营
- 人才盘点
- 继任分析
- 数据健康
- 汇报报告

其中：

- 首页 / 总览 / 汇报报告：主讲页
- 组织运营 / 人才盘点 / 继任分析 / 数据健康：证据页

## 双演示样本

### 官方案例

- 用于首页、总览与汇报报告
- 数据口径统一，不故意注入脏值
- 附带 2023-01 至 2025-12 的组织与招聘轨迹

### AI 清洗演示版

- 用于上传演示和数据健康页
- 保留字段别名、部门别名、岗位简称、缺失与异常
- 组织故事与官方案例一致

## 数据资产

当前 Demo 包含四层数据：

1. `employee_master`
2. `workforce_monthly`
3. `recruiting_reqs`
4. `exit_events`

## 官方案例公司

公司名：`澄曜云服科技`

业务定位：

- 面向中大型企业的 B2B SaaS + 交付服务公司
- 收入结构包括 SaaS 订阅、实施项目、存量续费与增购

核心组织故事：

- 研发中心：高潜储备不低，但 ready-now 管理梯队偏薄
- 销售增长：当前业绩强，但头部依赖重、保留风险高
- 交付运营：运行稳定，但成长动能不足、后备薄
- 产品与设计：关键岗位集中，单点依赖明显
- 客户成功：团队规模不小，但高潜识别不足
- 人力资源 / 财务法务 / IT与数据平台：人数少但继任暴露高

## 三年演化主线

### 2023

- 扩张期
- headcount 增长快
- 招聘量高
- 研发和产品扩编
- 管理梯队开始跟不上

### 2024

- 销售冲刺期
- 销售业绩上来
- regrettable attrition 上升
- 招聘成本抬高
- 客户成功与交付压力增加

### 2025

- 收敛与优化期
- 总人数增速放缓
- 运营稳定但停滞
- 支持职能继任空窗暴露
- 招聘周期回落，但内部供给仍不足
